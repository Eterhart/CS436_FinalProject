**ภาพรวมระบบ**

แอปนี้เป็นเว็บแอปแบบหน้าเดียว (Single Page Application) ที่สร้างด้วย React + TypeScript และใช้ Vite เป็นเครื่องมือบิลด์ จุดประสงค์คือช่วยให้ผู้ใช้สร้างรหัสผ่านหรือวลีผ่านตัวเลือกต่าง ๆ และดูประวัติรหัสผ่าน รวมถึงประเมินความแข็งแกร่งของรหัสผ่าน

---

**ข้อกำหนดของระบบ**

- Git (ถ้าต้องการ clone โค้ด)
- Node.js 18+ และ npm
- Windows / macOS / Linux ที่ติดตั้งเบราว์เซอร์สมัยใหม่ (Chrome, Edge, Firefox)

---

**ส่วนติดต่อผู้ใช้ (UI) — คำอธิบายและวิธีใช้งานทีละขั้นตอน**

1) สร้างรหัสผ่าน (Password Options)

- เลือกความยาวของรหัสผ่าน: ปรับสไลเดอร์หรือป้อนตัวเลข
- เลือกตัวเลือกตัวอักษร: ตัวพิมพ์ใหญ่, ตัวพิมพ์เล็ก, ตัวเลข, สัญลักษณ์
- ปุ่ม "Generate" จะสุ่มรหัสผ่านตามเงื่อนไข
- ผลลัพธ์จะปรากฏที่กล่อง `PasswordDisplay` และมีปุ่มคัดลอก

3) สร้างวลี (Passphrase Options)

- เลือกจำนวนคำและตัวคั่น (separator)
- สามารถเลือกตัวเลือกคำเพื่อความจำง่าย (เช่น รวมคำไทย/อังกฤษ ขึ้นกับการตั้งค่า)

4) แสดงผลและคัดลอกรหัสผ่าน (PasswordDisplay)

- กล่องแสดงรหัสผ่านที่สร้าง
- ปุ่ม "Copy" เพื่อคัดลอกไปยังคลิปบอร์ด

5) ประวัติรหัสผ่าน (PasswordHistory)

- แสดงรายการรหัสผ่านที่เคยสร้างและเวลาที่สร้าง
- มีตัวเลือกในการลบหรือคัดลอกรายการจากประวัติ

6) คะแนนความปลอดภัย (SecurityScore)

- แสดงระดับคะแนนของรหัสผ่านที่สร้างขึ้น (เช่น ต่ำ/ปานกลาง/สูง)
- ให้คำแนะนำเพื่อปรับปรุงความแข็งแกร่ง เช่น เพิ่มความยาวหรือใช้สัญลักษณ์

---

**ฟีเจอร์หลักและตัวอย่างการใช้งาน**

- สร้างรหัสผ่านแบบสุ่มตามเงื่อนไขที่เลือก
- สร้าง passphrase ที่จดจำง่ายแต่ปลอดภัย
- คัดลอกไปยังคลิปบอร์ดด้วยคลิกเดียว
- ประเมินและแสดงคะแนนความแข็งแกร่งของรหัสผ่าน
- เก็บประวัติการสร้าง (ถ้ามีการเชื่อมต่อกับ storage)

---

**การแก้ไขปัญหาเบื้องต้น (Troubleshooting)**

- ถ้าแอปไม่เริ่ม:
  - ตรวจสอบว่า `node` และ `npm` ติดตั้งอยู่
  - รัน `npm install` อีกครั้ง และตรวจสอบ error ที่แสดง
- ถ้าหน้าจอว่างเป็นสีขาว:
  - เปิด DevTools (F12) ดู Console สำหรับ error
  - ตรวจสอบว่ามีการโหลด `main.tsx` และ `index.css`
- ถ้าฟีเจอร์บางอย่างไม่แสดง:
  - ตรวจสอบเวอร์ชันของเบราว์เซอร์
  - ตรวจสอบการตั้งค่า Tailwind/CSS (ไฟล์ `tailwind.config.js` และ `src/index.css`)
